# Travel eSIM

Keep the UK SIM and buy a travel esim (**Saily**, **Airalo**, ...) for data.
Keep calls on the UK number but for any data use the travel esim.

## Checklist
- [ ] Set up the travel SIM before leaving but leave UK SIM as default
- [ ] Data roaming is turned off for the UK SIM
- [ ] Default mobile data is set to the travel SIM
