## Hardware Keys
Something you have (e.g. YubiKey) + something you know (e.g. PIN) or something
you are (i.e. biometric).

## Trusted Platform Module (TPM)
Secure cryptoprocessor. Like a serial number that can't be spoofed and can also
do things like signing.

## Hardware Security Module (HSM)
Like TPM for server racks. Manages lots of keys.
